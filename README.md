## 🏛️ Filosofia da Arquitetura (O que está sob o capô?)

O projeto foi escrito do zero obedecendo o paradigma **Vanilla Pura**:
1. **Zero Frameworks:** Nenhuma biblioteca de JavaScript (como React, Vue ou JQuery) pesa no client-side. A lógica flui instintivamente pelo motor nativo do navegador.
2. **Identidade "Lotus JPS":** Uma paleta visual premium (Black & Gold / Dark Mode) mesclada à estética moderna do *Glassmorphism* (Vidro Fosco), garantindo contraste, imersão e elegância corporativa.
3. **Lightbox Carousel Nativo:** Construímos um sistema de Modal para visualização de certificados e mockups de interface totalmente "caseiro", suportando paginação via Setas, Click Outside, e Binding direto com teclas de atalho (ESC, setas direcionais), tudo escrito sem depender do pesado Bootstrap.

## 📦 Projetos Exibidos (A Trindade Full-Stack)

O portfólio faz a consolidação do tripé de conhecimentos:
1. **Infraestrutura e DevOps:** *Otimizador Ultimate de Windows* — Scripts pesados em Batch/PowerShell para Hardening e Debloat.
2. **Engenharia de Back-End:** *Monitor de Ativos (Python)* — Daemon assíncrono para raspagem, parsing de cotações em tempo real e notificações ao nível de Sistema Operacional (Plyer).
3. **UI / Front-End:** *Landing Page Customizada* — Um show de fluidez usando mosaicos dinâmicos, partículas caídas e iframes interativos codificados na unha.
