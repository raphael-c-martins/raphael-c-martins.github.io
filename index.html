Monitor de Preços de Skins em Tempo Real

Automação com Python

Um programa feito em Python que analisa preços na loja da Steam de forma automática o dia todo. Ele verifica os itens, converte os valores de Dólar para Real instantaneamente pegando a cotação do dia, salva um histórico completo no computador e dispara notificações visuais na tela avisando quando um item entra em promoção.

Landing Page Interativa (Front-End)

UI Design e Desenvolvimento Web

Um site altamente criativo focado em engajamento visual. O maior destaque técnico é um Envelope Interativo animado (desenvolvido na unha com CSS/JS): ao ser clicado, a aba do envelope se abre e uma carta desliza para a tela revelando o texto. O projeto também possui galeria de imagens dinâmica (estilo mosaico), sistema de partículas caindo no fundo da tela e até um player de música embutido!
